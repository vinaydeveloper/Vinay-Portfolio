By Vinay Khandelwal

Implemented secure authentication systems with sign-up and sign-in functionality using JWT and bcrypt

To ensure secure access to certain pages, the application implements authentication through sign-up and sign-in processes. On the sign-up page, users can create a new account by providing necessary details such as email, username, and password. After successful registration, they are redirected to the sign-in page where they can log in with their credentials. Upon logging in, the application stores an authentication token, allowing access to protected routes. If a user tries to access these protected routes without being logged in, they are automatically redirected to the sign-in page. Once authenticated, users can access protected content until they log out, which removes the token and secures the session.

The currency converter allows users to convert one currency to another in real time. On the main page, users select the source and target currencies, enter the amount to convert, and the application provides the converted value based on current exchange rates. The exchange rates are regularly updated to ensure accurate conversions. If a user enters an invalid currency or amount, the application will display an error message. Once the conversion is complete, users can view the result instantly, making it easy to understand the exchange value.
